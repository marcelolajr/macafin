# macafin
    is a small application towards dynamic generation of financial reports. 

## Installation
    easy peasy:
      1. Clone it!
      2. pip install flask (may need privileges i.e sudo)
      3. export FLASK_APP=path/to/macafin.py
      4. flask run
      4. Voil√° ;-)
